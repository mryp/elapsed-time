# elapsed-time

日時データを含むログレコード間の経過時間を計測する

## 開発環境

- .NET 5.0
- Windowns Forms

## 使い方

1. 経過計測したいログの開始と最後を含むデータを時間付ログに貼り付ける
2. 計算ボタンを押下する

## フォーマット

ログレコードは先頭行が日時フォーマットになっているCSVデータとする  
経過時間は最初と最後のデータのみを参照するので中間にデータがあっても無視する

### 例

```text
2020-11-30 12:48:11.116,INFO,"アプリケーション起動",""
2020-11-30 12:48:23.783,INFO,"アプリケーション終了",""
```

## 注意事項

- 現在のバージョンは大量のデータは貼り付けられないことがあります。（.NET 5 Windows Formsの不具合？）
